# Presentation Service Website (Demo)

This is a **front-end demo project** for a presentation ordering service.

## Important Notice

This website is **NOT a real payment system**.

- Payments are NOT processed
- "Pay Now" button is only for UI demonstration
- No backend or database is connected
- No real orders are stored or sent automatically

## Purpose

This project was built to practice:
- HTML
- CSS
- JavaScript
- UI design for a service-based website

## Features

- Order form UI
- Payment page UI
- Responsive design (basic)
- Demo interactions only

## Tech Stack

- HTML
- CSS
- JavaScript
